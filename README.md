## 📌 About the Project

**Netflix Clone** is a simple landing page inspired by Netflix.  
It includes a modern hero section, email input area, feature cards, FAQ section, and footer layout.

This project is created using **HTML** and **CSS**, with a small JavaScript snippet used for the FAQ collapsible section.
